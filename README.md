# LoL Highlight Frontend

리그 오브 레전드 전적 검색 및 AI 하이라이트 분석 웹 애플리케이션입니다.

## 주요 기능
*   **전적 검색**: 소환사 이름으로 최근 전적 조회
*   **상세 분석**: 매치 아이템, KDA, 딜량 그래프 제공
*   **AI 코칭**: 플레이 피드백 제공
*   **하이라이트**: 주요 장면 자동 생성 및 재생

## 기술 스택
*   **Frontend**: React, TypeScript, Vite
*   **Styling**: Tailwind CSS
*   **State Management**: Zustand, React Query

## 시작하기

### 설치
```bash
npm install
```

### 실행
```bash
npm run dev
```
